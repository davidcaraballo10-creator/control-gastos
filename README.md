App: Mis Gastos

Descripción:
- Aplicación web estática para registrar gastos personales.
- Guarda los datos en localStorage del navegador.

Cómo usar:
1. Abre `gastos-app/index.html` en tu navegador.
2. Rellena monto, categoría, fecha y descripción y pulsa "Guardar".
3. Edita o elimina entradas con los botones correspondientes.
4. Usa los filtros para acotar por fecha o categoría.

Notas:
- Los datos quedan en tu navegador (localStorage). Para borrar todo: abre la consola y ejecuta `localStorage.removeItem('mis_gastos_v1')` o borra desde las herramientas del navegador.
- Si quieres exportar/importar datos puedo añadir esa funcionalidad.
